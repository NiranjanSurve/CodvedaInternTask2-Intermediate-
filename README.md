## Internship Task
Level 2 – Intermediate  
Codveda Technologies – Web Development Internship

## Description
A professional To-Do List application built using HTML, CSS, and JavaScript.
The app allows users to add, edit, delete, and complete tasks, with completed
tasks displayed separately as previous tasks. Data is persisted using browser
local storage.

## Features
- Add new tasks
- Edit existing tasks
- Mark tasks as complete
- Delete tasks
- Separate section for completed (previous) tasks
- Automatic data persistence using Local Storage
- Clean and user-friendly UI

## Technologies Used
- HTML5
- CSS3
- JavaScript (ES6)
- Browser Local Storage

## Learning Outcomes
- DOM manipulation
- Event handling
- CRUD operations
- Local storage state management
- UI/UX improvement through task separation
